rnaseqlib: a simple, lightweight pipeline for RNA-Seq analysis

rnaseqlib is a pipeline for processing RNA-Seq data. It currently supports:

  • mRNA-Seq
  • Ribo-Seq: Ribosome profiling/footprinting
  • CLIP-Seq

It works for any standard UCSC genome. Please see rnaseqlib.readthedocs.org for details and documentations.
